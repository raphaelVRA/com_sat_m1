Reception_gestion_MQTT

Ce document explique comment configurer et utiliser le fichier Reception_gestion_MQTT.json pour gérer la réception et la gestion des messages MQTT avec des alertes via Telegram.

Prérequis:
- Une ou plusieurs cartes Echostar_term_7.
- Un compte Telegram.

Configuration :
 1- Renseigner les informations spécifiques à votre module Echostar_term_7
 2- Rentrer le token Telegram de l'appareil qui doit recevoir les notifications d'alerte du risque incendie
 3- Régler le temps entre chaque test de risque incendie
